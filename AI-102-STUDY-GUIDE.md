The AI-102 exam measures your ability to accomplish the following technical tasks:

### 1. Plan and Manage an Azure AI Solution (15-20%)
   - Select appropriate Azure AI services
   - Plan for security and compliance
   - Manage Azure AI service resources

### 2. Implement Computer Vision Solutions (15-20%)
   - Analyze images and videos
   - Implement Custom Vision solutions
   - Detect, analyze, and recognize faces

### 3. Implement Natural Language Processing Solutions (30-35%)
   - Analyze text
   - Process speech
   - Translate text and speech
   - Build conversational AI solutions
   - Create question answering solutions

### 4. Implement Knowledge Mining and Document Intelligence Solutions (10-15%)
   - Implement Azure AI Search solutions
   - Extract data from forms and documents

### 5. Implement Generative AI Solutions (10-15%)
   - Use Azure OpenAI Service
   - Optimize generative AI solutions

### Exam Day Tips (from Microsoft Learn)

**Question Types to Expect:**
- **Multiple Choice** - Select one correct answer
- **Multiple Answer** - Select all correct answers (2-4 typically)
- **Drag and Drop** - Match items or order steps
- **Case Studies** - Multi-question scenarios
- **Hot Area** - Click on correct part of image/diagram
- **Review Screen** - Flag questions to review later

**Time Management:**
- 100-120 minutes for ~60 questions
- ~2 minutes per question average
- Flag difficult questions, move on, return later
- Case studies take longer - allocate extra time

**Common Traps:**
- "Most appropriate" vs "possible" solutions
- Reading too fast and missing keywords like "NOT"
- Overthinking simple questions
- Ignoring resource constraints in scenarios
